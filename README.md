# chat-with-web
chat-with-web

# build the image
```
docker build -t chat-with-web .
```

# run the image
```
docker run -p 5000:80 chat-with-web
```

# run the image in background
```
docker run -d -p 5000:80 chat-with-web
```

# go to the browser and type
```
http://localhost:5000
```
